DMS Integration

Reference how each DMS shares data with myKaarma so you can quickly explain timing, behavior, and any gaps.

DMS Integration Matrix

CDK Drive
Push + Pull
  • Customers & Vehicles

    Nightly sync of customers and vehicles. New records appear automatically in myKaarma.

  • RO Creation / Updates

    Open ROs and status changes are updated on a scheduled pull throughout the day.

  • MPI / Line Item Writeback

    Approved lines can be written back to CDK from ServiceCart using the integration button.

Reynolds & Reynolds
Push + Pull
  • Customers & Vehicles

    Customer and vehicle data sync automatically via scheduled updates.

  • RO Header Data

    RO details update when the RO is active inside Reynolds.

  • Line Items / Pricing

    Some pricing/line data syncs, while other detail fields remain limited or view-only.

Dealertrack
Push + Pull
  • Appointments

    The scheduler reads Dealertrack appointments and syncs availability.

  • ROs / Status

    Open ROs update throughout the day for communication and MPI workflows.

  • Writeback

    ServiceCart-approved work can be written back through the integration.

DealerBuilt
Pull Focused
  • Customer / Vehicle Data

    Automatic pull of customer and vehicle info for communication and RO lookup.

  • RO Data

    Open RO details sync for messaging, media, and payment workflows.

  • Writeback

    No writeback; ServiceCart is presentation only.

Automate
Pull Focused
  • Customers / Vehicles

    Automatic pull for customer and vehicle data updates.

  • Appointments

    Scheduler syncs appointments directly from Automate.

  • RO Writeback

    No writeback—line edits must be done in the DMS.

PBS
Pull Focused
  • Customers & Vehicles

    Nightly automatic pull keeps customer and vehicle data updated.

  • RO Header & Status

    ROs sync for messaging, approvals, and payments.

  • Writeback

    No writeback; edits must be completed inside PBS.

Tekion
Modern API
  • Real-time APIs

    Real-time updates for customers, vehicles, and repair orders.

  • Appointments

    Scheduler syncs Tekion appointments with minimal delay.

  • Line Item Handling

    Some line-level operations are display-only depending on integration scope.
